# Project 6
Building a JSON API

## Directions
The routes are as follows

Root: /

Get: /name (name refers to the Country itself so if you would like to find Japan,Congo,etc... you would type /Japan after the normal url)

Post: /

Put: country/name

Delete: /delete/name
